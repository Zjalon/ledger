# 账本页面重新设计

## 目标

简化账本页面（Home.vue）的信息层级，移除冗余的概览区域，将筛选功能集中到筛选弹窗中，并在流水卡片中展示账户信息。

## 变更概述

| 操作 | 内容 |
|------|------|
| 移除 | 日期导航栏（‹ prev \| 日期 \| next › + 日/周/月 tabs） |
| 移除 | Hero 成员收支概览卡片（各成员收入/支出汇总表格） |
| 新增 | 筛选弹窗中增加日历日期范围选择器 |
| 新增 | 筛选弹窗中增加成员多选筛选 |
| 新增 | 流水卡片中显示账户信息 |
| 保持 | 搜索栏入口、关键词筛选、金额范围筛选、类型筛选 |

## 页面布局（简化后）

从上到下：

1. **Ledger 外壳顶栏** — 品牌 + 同步状态（不变）
2. **搜索/筛选入口栏** — "搜索流水" 按钮，有活跃筛选时显示角标
3. **分区标题** — 默认 "今日流水"，有筛选时 "流水 · N 条"
4. **流水卡片列表** — 可滚动，左滑编辑/删除

默认展示今日流水（不弹出筛选弹窗）。

## 筛选弹窗（增强后）

从上到下的筛选项：

### 1. 日期范围（新增）

使用 Vant 内置的 `van-calendar` 组件，`type="range"` 模式。

- 默认不选中日期（重置状态 = 不限日期，展示全部流水）
- 点击日期区域打开日历弹窗，选择起止范围
- 选中后筛选按钮旁显示日期摘要（如 "5月14日 – 5月18日"）
- 重置筛选时清空日期范围

### 2. 成员筛选（新增）

多选 chips，从账本的 users 列表中读取成员。

- 展示所有成员昵称，点击切换选中/取消
- 多选为 AND 关系（选中 "张三" + "李四" = 只显示这两人）
- 未选中任何成员 = 不限，显示所有人的流水

### 3. 关键词（已有）

搜索备注内容。

### 4. 金额范围（已有）

最小/最大金额输入框。

### 5. 类型（已有）

支出/收入/转账 chips 多选。

### 底部按钮

"查看 N 条结果"，点击关闭弹窗应用筛选。

## 流水卡片 — 账户信息（新增）

### 支出/收入

在分类备注行下方，金额行上方，增加一行账户标签：

```
成员名          支出     HH:mm
分类 · 备注
账户名
              − ¥32.00
```

- 标签样式：灰色小标签，灰色背景，圆角
- 直接显示账户名称（如 "招商银行"、"现金"），不使用 icon 字段
- 没有关联账户的旧数据（`accountId` 为空）：不显示标签行，布局不变

### 转账

账户标签行显示转出→转入：

```
成员名          转账     HH:mm
备注（可选）
转出账户  →  转入账户
              ¥500.00
```

- 使用箭头连接两个账户标签
- `transferTo` 字段为空时只显示转出账户

## 数据依赖

- 账户列表：通过 `ep.tableGetAllItems(bookId, "accounts")` 获取，已有
- 成员列表：通过 `ep.tableGetAllItems(bookId, "users")` 获取，已有
- Transaction 的 `accountId` 和 `transferTo` 字段：已定义，部分旧数据可能为空
